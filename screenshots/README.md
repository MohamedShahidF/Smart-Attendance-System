# Smart Attendance System

## Overview
A web-based attendance management system that allows students to mark attendance only when connected to the same local network as the teacher.

## Features
- Secure attendance marking
- Local network verification
- Student attendance tracking
- Attendance report generation
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js

## Problem Solved
Traditional online attendance systems allow students to mark attendance remotely. This project ensures that attendance can only be marked when students are physically present and connected to the teacher's network.
